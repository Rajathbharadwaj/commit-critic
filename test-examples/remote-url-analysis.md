# Remote URL Analysis

Analysis of the [anthropics/anthropic-cookbook](https://github.com/anthropics/anthropic-cookbook) repository (last 5 commits).

## Command

```bash
python commit_critic.py --analyze --url "https://github.com/anthropics/anthropic-cookbook" --limit 5
```

Or via LangGraph API:
```bash
curl -X POST "http://localhost:2024/threads/THREAD_ID/runs" \
  -H "Content-Type: application/json" \
  -d '{"assistant_id":"commit_critic","input":{"messages":[{"role":"user","content":"Analyze the last 5 commits from https://github.com/anthropics/anthropic-cookbook"}]}}'
```
